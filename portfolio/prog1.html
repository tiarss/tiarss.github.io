Tugas Akhir : Kontrol Kecepatan Robot Kapal untuk Kestabilan Heading menggunakan Inertial Measurement Unit

2019 s/d 2020

Perancangan sistem kontrol kecepatan robot kapal terhadap sudut heading kapal menggunakan sensor IMU yang akan dikerjakan pada proyek akhir ini merupakan sebuah sistem dimana robot kapal dapat menyesuaikan arah kapalnya sesuai dengan setting arah kapal di awal yang menggunakan dari sensor yang ada pada modul IMU, MPU 9250. Sistem ini dilengkapi dengan 1 modul IMU yang dipasang pada bagian tengah kapal. Data pada sensor digunakan untuk proses kontrol pada daun kemudi kapal yang dikendalikan dengan Motor Servo DS3218 dengan sudut pergerakan 120 derajat. Sehingga jika terjadi error pada sudut orientasi heading maka servo akan bergerak untuk mengembalikan arah sudut kapal pada set point awal. Kapal akan dikendalikan oleh Remote Control melalui aplikasi android yang terhubung melalui WiFi. Penggerak dari kapal ini adalah Motor BLDC. Motor BLDC akan berfungsi untuk mempercepat perbaikan heading kapal ke sudut setpoint nya. Berikut merupakan hasil implementasi perancangan sistem

Pada blok diagram diatas, terdapat beberapa komponen penting. Seperti yang telah di jelaskan sebelumnya. Berikut merupakan Keterangan bagian bagian pada blok diagram :

  1. Input yang digunakan adalah IMU MPU9250, berfungsi sebagai pengirim data 3 axis accelerometer, 3 axis gyroscope dan 3 axis magnetometer yang selanjutnya akan diolah oleh microcontroller. Selain itu input juga berasal dari ESP-01 dan Smartphone dengan aplikasi android. Keduanya berperan dalam pengiriman data perintah kepada mikrokontroller sebagai remote control. Tidak lupa juga Catu daya sebagai input bagi mikrokontroller dan Motor BLDC.
  2. Mikrokontroller STM32F4 Discovery sebagai tempat memproses data input hingga diproses juga sebagai data output.
  3. Output berupa actuator yaitu servo sebagai pengendali arah kapal dan motor bldc sebagai pengendali gerak kapal.
